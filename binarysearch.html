Randomize Array
  1. Start by initializing the lower bound of the search interval to the first element of the array and the upper bound to the last element.
  2. Find the middle element of the array by calculating the average of the lower and upper bounds (mid = (low + high) / 2).
  3. Compare the middle element with the target value.
    • If the middle element is equal to the target value, return its index.
    • If the target value is less than the middle element, set the upper bound to be one element before the middle element.
    • If the target value is greater than the middle element, set the lower bound to be one element after the middle element.
  4. Repeat steps 2 and 3 until the target value is found or the lower bound becomes greater than the upper bound, indicating that the target value is not in the array.
  5. If the target value is found, return its index. If it is not found, return -1 to indicate that the target value is not in the array.
Note that this algorithm works only when the input array is sorted.
